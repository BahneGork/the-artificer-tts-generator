üìã What is VB-CABLE?

VB-CABLE is a virtual audio cable that allows you to route audio between applications on Windows.

For The Artificer, VB-CABLE enables the "Send to Discord" feature, which lets you play NPC voices directly into your Discord voice chat during live TTRPG sessions.

How It Works:

  1. VB-CABLE creates a virtual microphone on your computer
  2. The Artificer plays audio into this virtual microphone
  3. Discord picks it up as if you were speaking into a real mic
  4. Your party hears the NPC voice in real-time!

üîç Troubleshooting

Problem: "No Virtual Cable Found" Error

Solution:

  1. Verify VB-CABLE is installed (check Sound Settings ‚Üí Recording)
  2. If not installed, follow Step 2 above
  3. Make sure you restarted after installing VB-CABLE
  4. Try the üîß Reset Audio Device button in The Artificer

Problem: Discord Doesn't Hear The Audio

Solution:

  1. Check Discord ‚Üí Settings ‚Üí Voice & Video
  2. Ensure Input Device is set to "Default"
  3. Disable "Automatically determine input sensitivity"
  4. Try leaving and rejoining the voice channel

Problem: Microphone Stays on Virtual Cable

Solution:

  1. Click üîß Reset Audio Device in The Artificer
  2. Or manually change default microphone in Windows Sound Settings
  3. Close and restart The Artificer if needed

Problem: Audio Quality Issues

Solution:

  1. Lower Discord voice bitrate if audio is choppy
  2. Disable unnecessary audio effects in The Artificer
  3. Check your internet connection
  4. Make sure no other apps are using VB-CABLE

‚öôÔ∏è Advanced: Manual Setup (Optional)

If you want to route audio manually without using The Artificer's automatic switching:

Manual Routing Steps:

  1. Play audio normally in The Artificer (Preview button)
  2. Manually set Windows default microphone to "CABLE Output"
  3. Discord will hear whatever plays through VB-CABLE
  4. Manually switch back to your real microphone when done

This approach gives you more control but requires manual device switching.
